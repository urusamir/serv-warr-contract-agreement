# Kavak Aftersales Service Report — Project Instructions

Before implementing any task, ask clarifying questions until you are 95% confident about what the user wants and what needs to be executed. Do not begin coding or making changes until that confidence threshold is reached.

## Project Overview
React + TypeScript SPA (Vite) for generating Kavak automotive service reports. Reports are submitted to Supabase (table: `service_reports`, storage bucket: `service-report-pdfs`). PDF generation is client-side via jsPDF.
